When we need to store several related items we can use objects.
Example: 

          
            let member = {name:"Jack", lastName:"Brown", gender:"male"};

Spaces and line breaks can be used inside an object.

          
            let member = {  
              name    :"Jack", 
              lastName:"Brown", 
              gender  :"male"
            };

As a property a key, a string or a number can be used.
And a value can be ANYTHING ! 

          
            let member = {  
              name    :"Jack", 
              lastName:"Brown", 
              age     :22,
              items   :["item1", "item2"]
              misc    :function(...) {...},
            };

As in the example below each "name: value" blocks named as "property". 

          
            name    :"Jack",  // A property
            lastName:"Brown", // Another property

To access a object property: 

          
            objectName.propertyName
            //or
            objectName["propertyName"]

Functions inside an objects are called "methods".

          
            let member = {  
              name    :"Jack", 
              lastName:"Brown", 
              gender  :"male",
              fullName:function() {
                return this.name + " " + this.lastName;
              }
            };
