							#chessboard  {

								border: 2px solid black;
								height: 70px;

							}
							
							#chessboard tr:nth-child(odd) td:nth-child(even) {
								background: black;
							}
							#chessboard tr:nth-child(even) td:nth-child(odd) {
								background: black;
							}
							#chessboard tr:nth-child(even) td:nth-child(even) {
								background: white;
							}
							#chessboard tr:nth-child(odd) td:nth-child(odd) {
								background: white;
							}
							
							td {
								color: brown;
								text-align: center;

							}

References:

1) W3Schools CSS border-collapse Property

border-collapse may take two values: -separate: Default one. Each cell has its own border. -collapse: Collapses the borders of neighbour cells.

2) Even and Odd Rules

To choose even numbered childs: td:nth-child(even); To choose odd numbered childs: td:nth-child(odd)
