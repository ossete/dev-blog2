JavaScript Arrays

JavaScript Arrays: description and code examples

PUBLISHED APRIL 2020 BY OSSETE

When we need to store several related items we can use arrays.
Example: 

          
            let sizes = ["large", "medium", "small"];

Array items are given an “index number” and computer indexes start with “0”. To change the value of an item / index:

          
            sizes[0] = "big";

Spaces and line breaks can be used inside an array.

          
            let sizes = [
              "large", 
              "medium", 
              "small"
            ];

An array item can be a function, an object or another array.

          
            let arr = [ 
            theFunction, 
            {color: "red", signal: "1"},
            ["red", "green", "blue"]
            ];

Arrays can be used with built-in array properties.

          
            let sizes = ["large", "medium", "small"];
            let itemAmount = sizes.length;  // length of sizes array is 3

New elements can be added to arrays.

          
            let sizes = ["large", "medium", "small"];
            sizes.push("xsmall");  // adds a new element (xsmall) to sizes

To remove the last element pop() method is used.

          
            let sizes = ["large", "medium", "small"];
            sizes.pop(); // removes "small"

To remove the first element shift() method is used.

          
            let sizes = ["large", "medium", "small"];
            sizes.shift(); // removes "large"

To remove and add new elements splice() method can be used.

          
            let sizes = ["large", "medium", "small"];
            sizes.splice(0, 0, "xlarge"); // from 0. index, delete 0 item, add "xlarge"
